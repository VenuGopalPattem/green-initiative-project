# The Green Trees Initiative 

This project is a fully responsive website for **The Green Trees Initiative**, a non-profit organization focused on addressing climate change and empowering communities in developing nations. The site highlights key issues, showcases initiatives, shares stories, promotes events, and provides ways for users to take action.

---

## Features

### Fully Responsive Layout  
Built mobile-first and optimized for **tablet** and **desktop** using CSS Flexbox, Grid, and structured media queries.

### Multi-Page Website  
Includes:
- **Home Page**
- **Donation Page**

###  Unified Design System  
The project uses a consistent visual system with:
- Global utility classes (flex, grid, spacing helpers)
- Brand colors (navy, green, white)
- Flood font for headers
- Reusable CTA buttons and section labels

### Consolidated Stylesheets  
Each page uses a dedicated stylesheet:
- `style.css`  
- `donation.css`  

---

## Donation Page  
The donation screen features a structured, responsive form including:
- Personal information fields  
- Payment information fields  
- Styled number inputs

---

## Technology Used  
- **HTML5**  
- **CSS3**  
- **Adobe Typekit (Flood Font)**  
- Responsive design principles using Flexbox & Grid
